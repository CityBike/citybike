# CityBikeGirl

A “runner” game about a hero riding her bike to school and all the obstacles she must avoid to succeed! Built in Phaser.js with the help of Tiled, TexturePacker and Photoshop. Developed in 4 days as a Stackathon project at Fullstack Academy.
